# Contributing to Awesome Content Creation

Thank you for wanting to improve this list! Here's how to contribute.

## Guidelines

### Adding a Resource

1. **Check for duplicates** — Search the existing list first.
2. **Format** — Follow the existing format: `[Resource Name](URL) — Brief description.`
3. **Quality** — Only add resources that are actively maintained and genuinely useful.
4. **No affiliate links** — Unless the resource itself is an affiliate program.
5. **Descriptions** — Keep them brief (under 20 words), objective, and informative.
6. **Alphabetical order** — Add entries in alphabetical order within their category.

### Suggesting a New Category

If you think the list needs a new category:

1. Open an issue first to discuss it
2. A new category should have at least 3-5 entries to justify its existence
3. Explain why existing categories don't cover it

### Removing a Resource

If a resource is:
- Dead or abandoned (site down, no updates in 2+ years)
- Acquired and redirected
- No longer relevant

Open a PR to remove it with a brief explanation.

## How to Submit

1. Fork this repository
2. Create a branch: `git checkout -b add-resource-name`
3. Make your changes
4. Submit a pull request with a clear title and description

## Code of Conduct

Be respectful. This is a community resource. We welcome all contributors regardless of experience level.
